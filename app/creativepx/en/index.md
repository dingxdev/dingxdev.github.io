# CreativePX App

A simple yet powerful pixel art creation tool.  

You can use any element as a pixel, including shapes, text, emojis, symbols, and images.  
Any pixel color can be chosen.  
Canvas size is flexible, up to 512x512.  
Supports multiple layers.  
Works on iPhone, iPad, and MacBook, with optional iCloud syncing.
